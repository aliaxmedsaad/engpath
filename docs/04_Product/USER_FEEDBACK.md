---

## How to Use This Log

When feedback is received (via direct message, email, a session, or any other channel), add a row with:

- **Date** — when the feedback was received
- **User** — first name or anonymous identifier (do not log personal data)
- **Feedback** — brief summary of what was said
- **Severity** — Critical / High / Medium / Low / Enhancement
- **Action Taken** — either the fix applied, a link to a known issue, or "No action — noted"

---

## Severity Guide

| Level | Meaning |
|---|---|
| Critical | Prevents core use of the product |
| High | Significantly impairs the experience |
| Medium | Noticeable issue but workaround exists |
| Low | Minor annoyance |
| Enhancement | Suggestion for improvement, not a bug |
